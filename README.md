## Project Overview

**QuickMCQs** is a free online platform designed for students and job aspirants in Pakistan to practice multiple-choice questions (MCQs). Users can attempt random MCQ tests, create their own tests, and share them easily with others.  

This project aims to make **exam preparation accessible, simple, and effective** without any registration or cost.

---

## Features

- Attempt **random MCQ tests** by category  
- **Create custom MCQ tests** and share with others  
- **Timed tests** to simulate real exam conditions  
- **Instant results** with correct answers for learning  
- Mobile-friendly and clean interface  

---

## Technologies Used

- **Frontend & Backend:** Next.js (React + Node.js)  
- **Styling:** Tailwind CSS  
- **Database:** MongoDB Atlas  
- **Icons:** Lucide Icons  
- **Deployment:** Vercel  
- **Version Control:** Git + GitHub
